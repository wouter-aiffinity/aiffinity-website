## 🎬 Animation Details

### Page Load
1. Hero content fades in
2. Neural network starts animating
3. Scroll indicator bounces

### Scroll Animations
- Cards fade in and slide up when 10% visible
- Staggered timing for grouped elements
- Smooth easing functions

### Hover Effects
- **Cards:** Lift 10px, shadow increases
- **Buttons:** Lift 3px, color brightens, shadow glows
- **Links:** Color shifts to cyan, underline animates

### Neural Network
- Nodes move continuously at 0.5px/frame
- Lines update every frame
- Opacity fades based on distance
- 60 FPS smooth animation
