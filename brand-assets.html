🚀 Quick Start

1. Open the Website

# Simply open index.html in your browser
open index.html

# Or use a local server
python -m http.server 8000
# Visit http://localhost:8000

2. Customize Content

Edit the HTML content in index.html to match your needs. All text is clearly marked with semantic HTML tags.

3. Adjust Colors

Modify the CSS variables in css/style.css at the top of the file to change the entire color scheme.

4. Deploy

Upload to any static hosting provider:

  • Netlify (drag & drop)
  • Vercel (GitHub integration)
  • GitHub Pages (free hosting)
  • Any traditional web host
