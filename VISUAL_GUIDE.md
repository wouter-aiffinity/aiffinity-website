## 🎭 Interactive Features

### Animations
1. **Scroll Animations:**
   - All cards fade in and slide up when entering viewport
   - Triggered by Intersection Observer API

2. **Neural Network:**
   - 50 cyan nodes moving randomly
   - Lines connect nodes within 150px distance
   - Line opacity based on distance
   - Continuous animation loop

3. **Hover Effects:**
   - Cards: Lift up + shadow increase
   - Buttons: Scale, shadow, color shift
   - Links: Underline animation, color change

4. **Navigation:**
   - Smooth scroll to sections
   - Navbar shadow appears on scroll
   - Mobile menu slides from left

### Responsive Breakpoints
- **Desktop:** 968px+ (full layout)
- **Tablet:** 768px - 968px (adjusted grids)
- **Mobile:** < 768px (stacked layout, hamburger menu)
